Swap
Swap is a new take on the classic tile-based puzzle game, developed over the course of a month by my friend and I for the 2013 Github Game Off. At the end of the contest, the judges selected our game to win the grand prize. Swap is written in Javascript using the HTML 5 canvas for rendering.

Hextris
Hextris is a fast-paced puzzle game inspired by Tetris. My teammates and I developed this game at the HackExeter hackathon, where it won first prize. In subsequent months, the game received critical acclaim and has been downloaded and played by millions of users on the web and on Android and iOS devices.

Flare
Flare is a novel communications protocol that encodes data as a sequence of colors, which are then flashed on one phone's screen and read by the camera of another phone. My teammates and I developed Flare at the PennApps X collegiate hackathon, where we came in the Top 10 and received the Best iOS app prize.

Snapchat Keeper
Snapchat Keeper is an Android app (requiring a rooted phone) that allows a user to secretly save images sent via Snapchat. While on the Google Play store, the app received tens of thousands of downloads before being removed for trademark reasons. I developed it as a demonstration of the lack of security found in apps such as Snapchat.

Onion or Not?
Onion or Not is a simple game I developed in order to learn more about web development. It displays an Onion headline or a ridiculous but true headline curated by reddit.com/r/nottheonion, and then challenges the user to guess if it's an Onion headline.

Revisely
Revisely is a web application developed by a friend and I while following the Stanford MOOC "Startup Engineering" on Coursera. Revisely, our final project, is an application that allows users to post papers and received annotated feedback from other users, with a point system to encourage giving feedback.
